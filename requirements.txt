#   - Nemotron OCR v2 compiles a C++ CUDA extension (needs conda `aiml` env)
#   - Florence-2 requires transformers 4.40.0 (needs conda `florencetf` env)
#   - All other models run in the main `.venv` with transformers 5.x

# Per-environment install files:
#   pip install -r requirements.txt          # .venv (main)
#   pip install -r requirements-aiml.txt      # conda activate aiml first
#   pip install -r requirements-florencetf.txt # conda activate florencetf first
# ============================================================================
